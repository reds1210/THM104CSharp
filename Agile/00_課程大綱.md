# 敏捷開發實戰課程大綱 (Agile Development for Capstone Projects)

本課程旨在帶領學生從傳統瀑布式開發思維轉型為敏捷思維，並直接將敏捷工具應用於畢業專題。

## 課程模組結構

### [01. 瀑布式開發詳解 (Waterfall Model)](./01_瀑布式開發_Waterfall.md)
*   **理論**: 軟體生命週期 (SDLC) 六大階段詳解、文件導向與其局限性。
*   **實作**: 模擬圖書館系統在瀑布式下的風險。

### [02. 敏捷思維與宣言 (Agile Mindset)](./02_敏捷思維與宣言_Agile_Mindset.md)
*   **理論**: 敏捷宣言 4 大價值、12 條原則、VUCA 環境分析。
*   **實作**: 建立團隊溝通契約與核心價值排序。

### [03. 用戶故事與估算 (User Stories & Estimation)](./03_用戶故事與估算_User_Stories_Estimation.md)
*   **理論**: INVEST 原則、費波那契數列估算、規劃撲克。
*   **實作**: 撰寫專題 Product Backlog 並進行故事點估算。

### [04. 用戶故事地圖 (User Story Mapping)](./04_用戶故事地圖_User_Story_Mapping.md)
*   **理論**: 解決扁平化 Backlog 問題，視覺化使用者旅程。
*   **實作**: 繪製二維故事地圖，劃定 MVP (最小可行性產品) 界線。

### [05. Scrum 框架實戰 (Scrum Framework)](./05_Scrum框架_Scrum_Framework.md)
*   **理論**: 3-5-3 結構、KPT 回顧法。
*   **實作**: 啟動第一個 Sprint，進行衝刺規劃會議。

### [06. 看板方法與視覺化 (Kanban)](./06_看板方法_Kanban.md)
*   **理論**: 限制在製品 (WIP)、視覺化工作流。
*   **實作**: 建立 GitHub Projects 或 Trello 看板。

### [07. 敏捷技術實踐 (Technical Practices)](./07_敏捷技術實踐_Technical_Practices.md)
*   **理論**: CI/CD 觀念、Code Review、TDD 與結對編程。
*   **實作**: 設定 GitHub 分支保護規則與 PR 流程。

### [08. 專題實作整合指南 (Project Integration)](./08_專題實作指南_Project_Integration.md)
*   **實戰**: 學期 16 週的 Sprints 規劃建議、應對評審與老師的需求變更。
